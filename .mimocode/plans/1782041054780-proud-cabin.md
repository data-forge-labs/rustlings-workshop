### Tutorial structure (README.md)

1. **Preamble** — H1 + tagline, test-driven banner, `## What Is This Project?`
   - Python equivalent: lambdas + `functools`
   - Topics covered table: `Fn`/`FnMut`/`FnOnce`, `move` closures, custom `Iterator`, `IntoIterator`

2. **Body sections:**
   - §1 Introduction — why closure traits matter for data pipelines
   - §2 Prerequisites — references 12-RustIterators and 04-MasterMind
   - §3 Concept: The Three Closure Traits (`Fn`, `FnMut`, `FnOnce`)
     - Python comparison: lambdas have no trait distinction
     - ASCII diagram showing trait hierarchy: `FnMut: FnOnce`, `Fn: FnMut`
     - When each is required (thread spawn → FnOnce, sort_by → FnMut, filter → Fn)
   - §4 Concept: `move` Closures in Depth
     - Ownership transfer, when needed (threads, stored callbacks)
   - §5 Concept: Implementing `Iterator` for Custom Types
     - The `Iterator` trait contract
     - Example: `Counter`, `Fibonacci`, `FilterMap` custom iterators
   - §6 Concept: `IntoIterator` — Consuming Collections
     - `for` loop desugaring
     - Implementing `IntoIterator` for custom types
   - §7 Putting It All Together — build the lib.rs functions
   - §8 Exercises (Easy/Medium/Hard)
   - §9 Summary table
